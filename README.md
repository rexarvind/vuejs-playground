# vuejs-playground
Learning VueJS 3 in depth.

## Vue Directives
```
// bind with value to make reactive
v-bind:foo="bar" or :foo="bar"

// add event listeners
v-on:click="foo" or @click="foo"

// two-way data binding
<input v-model="message" />
```

- `v-bind:foo="bar"` or `:foo="bar"`
- `v-on:click="foo"` or `@click="foo"`
- `<input v-model="message" />`
